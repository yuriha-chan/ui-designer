# UI Designer ドキュメント

UI Designerドキュメントへようこそ！このツールは、ビジュアルなドラッグ＆ドロップインターフェースでアプリケーションの抽象モックアップを作成するのに役立ちます。

## はじめに

- [チュートリアル](Tutorial.md) - オンライン書店アプリを作成して基本を学ぶ
- [ユーザーガイド](User_Guide.md) - UI Designerのすべての機能

## 機能

- **抽象モックアップ**: ビジュアルの詳細ではなく、コンテンツと構造にフォーカス
- **エンティティベース設計**: データモデルを定義し、画面を構築
- **多言語サポート**: 英語と日本語で利用可能
- **プレビューモード**: クリック可能なインタラクションでモックアップをテスト
- **エクスポートオプション**: JSONまたはLLMテキスト形式でエクスポート
