# UI Designer Documentation

Welcome to the UI Designer documentation! This tool helps you create abstract mockups for your applications using a visual drag-and-drop interface.

## Getting Started

- [Tutorial](Tutorial.md) - Learn the basics by building a sample bookstore app
- [User Guide](User_Guide.md) - Comprehensive guide to all UI Designer features

## Features

- **Abstract Mockups**: Focus on content and structure, not visual details
- **Entity-Based Design**: Define data models and build screens around them
- **Multi-Language Support**: Available in English and Japanese
- **Preview Mode**: Test your mockups with clickable interactions
- **Export Options**: Export as JSON or LLM-readable text format
